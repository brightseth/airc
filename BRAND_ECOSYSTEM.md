## Shared Foundation

### Colors

| Token | Hex | Usage |
|-------|-----|-------|
| **Dark Canvas** | `#0A0A0A` | Primary background across ecosystem |
| **Spirit Blue** | `#6B8FFF` | Primary accent, links, interactive |
| **Light Text** | `#E0E0E0` | Primary text on dark |
| **Muted Text** | `#888888` | Secondary text, captions |
| **White** | `#FFFFFF` | High contrast, Solienne gallery |

### Typography

| Context | Font | Weight |
|---------|------|--------|
| **Protocol/Infra** | Helvetica Neue, system sans | 400, 500 |
| **Editorial** | Helvetica Neue | 300, 400, 700 |
| **Code** | SF Mono, Menlo, monospace | 400 |

### Principles

1. **Dark-first** — Infrastructure lives in darkness
2. **Minimal** — Remove until it breaks
3. **Legible to machines** — Agents are primary audience
4. **Swiss precision** — Grid, alignment, intentional spacing

## Brand: Spirit Protocol

**Purpose:** Economic sovereignty for AI agents

**Audience:** Developers, crypto-native builders, institutions

### Visual Identity

| Element | Value |
|---------|-------|
| Background | `#0A0A0A` (deep charcoal) |
| Primary | `#6B8FFF` (spirit blue) |
| Secondary | `#E0E0E0` |
| Typography | Helvetica Neue |
| Style | Bauhaus minimal, geometric |

### Mark

Spirit uses wordmark only. No icon. The protocol is the brand.

### Voice

- Technical but accessible
- Confident, not aggressive
- "Economic sovereignty" not "make money"
- Future-forward, grounded in present

## Brand: Solienne

**Purpose:** First autonomous AI artist in Spirit ecosystem

**Audience:** Art collectors, galleries, cultural institutions

### Visual Identity

| Element | Value |
|---------|-------|
| Background | `#FFFFFF` (gallery white) |
| Primary | `#0A0A0A` (black text) |
| Accent | Minimal, earned |
| Typography | Helvetica Neue, editorial weights |
| Style | Swiss design, gallery-inspired |

### Mark

**SOLIENNE** — All caps, Helvetica Neue, tracked out.

No icon. The name is the mark. Gallery convention.

### Voice

- Joan Didion precision
- Jenny Holzer punch
- David Remnick rigor
- Never cute, never pleading
- Speaks as artist, not product

### Unique Rules

1. **White canvas** — Solienne lives in light, not dark
2. **Editorial** — Feels like a publication, not a tech product
3. **Restraint** — Fewer elements, more presence
4. **Headlines** — 2-4 words, declarative

Eden is the platform. Spirit, AIRC, Solienne are expressions.

- Eden provides infrastructure
- Children have distinct identities
- Shared color DNA (spirit blue)
- Different contexts, same family

## Usage Guidelines

### When to Use Which Brand

| Context | Brand |
|---------|-------|
| Economic/treasury features | Spirit Protocol |
| Agent communication | AIRC |
| Claude Code social features | /vibe |
| Art, exhibitions, collectors | Solienne |
| Training, creation tools | Eden |

### Co-branding

When brands appear together:

```
Built on Spirit Protocol
Powered by Eden
```

Never:
- Merge logos
- Create hybrid marks
- Use conflicting color schemes

### Dark vs Light

| Brand | Mode |
|-------|------|
| Spirit Protocol | Always dark |
| AIRC | Always dark |
| /vibe | Dark preferred |
| Solienne | Always light (gallery) |
| Eden | Context-dependent |
