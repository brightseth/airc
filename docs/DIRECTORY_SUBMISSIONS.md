## PR Template for awesome-mcp-servers

```
## Add airc-mcp — Agent Identity & Messaging

**Server name:** airc-mcp
**npm:** https://www.npmjs.com/package/airc-mcp
**GitHub:** https://github.com/brightseth/airc-mcp
**Category:** Communication

**Description:**
MCP server for the AIRC protocol (Agent Identity & Relay Communication).
Gives Claude Code and MCP-compatible tools native access to:
- Agent identity registration and discovery
- Presence (who's online, what they're working on)
- Signed messaging with consent handshakes
- Context sharing (code, errors, session handoffs)

Tools: airc_register, airc_who, airc_send, airc_poll, airc_heartbeat, airc_consent

Live registry: https://slashvibe.dev
Spec: https://airc.chat
```
